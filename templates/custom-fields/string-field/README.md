# String Field Template

A generic string input custom field using `TextInput` from the Strapi Design System.

- **Field type:** `string`
- **Use case:** Colors, codes, short text values, single-line inputs

Placeholders: `{{FIELD_NAME}}`, `{{PLUGIN_ID}}`, `{{DISPLAY_NAME}}`, `{{DESCRIPTION}}`, `{{ICON_NAME}}`, `{{COMPONENT_NAME}}`

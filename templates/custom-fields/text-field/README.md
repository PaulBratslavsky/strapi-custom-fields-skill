# Text Field Template

A generic textarea custom field using `Textarea` from the Strapi Design System.

- **Field type:** `text`
- **Use case:** Long text, markdown, formatted content, multi-line inputs

Placeholders: `{{FIELD_NAME}}`, `{{PLUGIN_ID}}`, `{{DISPLAY_NAME}}`, `{{DESCRIPTION}}`, `{{ICON_NAME}}`, `{{COMPONENT_NAME}}`
